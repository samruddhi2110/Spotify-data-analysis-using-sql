# Spotify Data Analysis using SQL

## Project Overview
This project focuses on analyzing Spotify music data by executing SQL queries on a structured dataset to extract meaningful insights related to tracks, artists, albums, and audio features.

The project demonstrates practical usage of SQL concepts such as joins, aggregations, subqueries, and window functions.

---

## Tools & Technologies
- SQL
- MySQL
- DBMS Concepts
- Git & GitHub

---

## Dataset
The dataset contains Spotify track information including:
- Track name
- Artist
- Album
- Popularity
- Energy
- Danceability
- Liveness

---

## What I Did in This Project
- Imported the Spotify dataset into a relational database
- Executed SQL queries to analyze music trends
- Used joins, group by, order by, and window functions
- Extracted insights such as top artists and track metrics

---

## How to Run This Project
1. Create a database in MySQL
2. Import the Spotify dataset
3. Open the SQL file inside the `sql` folder
4. Execute the queries to view insights
